# Amazon Clone

A React-based Amazon clone project built to practice modern frontend development and e-commerce UI design. The project is structured as a Vite application and is intended to recreate the look and feel of an online shopping platform with a clean, responsive interface.

## Features

- Amazon-inspired shopping website layout
- React component-based frontend structure
- Fast development setup with Vite
- Responsive styling with CSS
- Organized project files for future backend and feature expansion

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

## Project Status

This project is currently in development. More e-commerce features such as product listings, cart functionality, search, authentication, and checkout flow can be added in future updates.
